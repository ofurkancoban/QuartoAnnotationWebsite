What is Quarto?

Quarto is the next generation of R Markdown — but language-agnostic and designed for the modern data-science ecosystem.

  • Renders .qmd files to HTML, PDF, Word, and more
  • Supports R, Python, Julia, and Observable JS
  • Built-in cross-references, citations & theming
  • Live preview with quarto preview

Key Features

🔁 Reproducibility

Code and narrative live together. Re-run documents to update figures and tables automatically.

🎨 Beautiful Output

Ship polished HTML reports, RevealJS slides, and LaTeX-quality PDFs from the same source.

🧩 Extensions

Community extensions add new formats, shortcodes, and filters to expand Quarto's capabilities.

🤝 Collaboration

Plain-text source files work seamlessly with Git and GitHub for version control and review.

Creating Presentations

Quarto presentations use RevealJS for web-based slides or Beamer for PDF slides, produced from the same .qmd source.

  • Divide slides with ## headings or horizontal rules ---
  • Add speaker notes inside ::: notes blocks
  • Animate fragments, transitions, backgrounds, and more
  • Embed live code outputs and interactive widgets
  • Export to PDF for distribution

Typical Workflow

  • Install Quarto CLI from quarto.org
  • Write your document in any text editor or IDE (VS Code, RStudio, Neovim …)
  • Run quarto render document.qmd to produce output
  • Use quarto publish to deploy to Quarto Pub, GitHub Pages, or Netlify
  • Iterate: edit source, preview changes instantly

Handy commands

quarto create project myproject
quarto preview myproject
quarto render myproject
quarto publish gh-pages
